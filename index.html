How does it work

An easy-to-use dashboard for all your APIs
you can run your APIs, see the results and even define scenarios for them.

Create
Collections

Create collaborative collections to add your APIs in them, add the people you want in them and assign them roles.

Run
Your API

In your dashboard, you have the ability to run your API requests with the attributes you like and see the resutls as well.

Create
Scenarios

You can create scenarios for the APIs in each of your collections, schedule them and so on.

Manage all your collections
in one place!

You can create unlimited amount of collections as you need, add your team to them, create requests in them, run and test your requests...

Basically you can do whatever you like!

Create and schedule
Scenarios!

After creating your collections, you can create scenarios for them and schedule them to run at specific times.

You can automate your reqeusts to run at the desired time.

Follow your friends!

You can easily search and find your friends.

Follow other people, run their API requests and much more.
